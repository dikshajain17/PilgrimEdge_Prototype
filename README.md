PilgrimEdge Prototype

Overview
This project demonstrates a prototype of an IoT-based smart wearable system for crowd safety.

Features
- SOS Alert System using Arduino
- Basic emergency alert mechanism

Tech Stack
- Arduino

 Note
The code has been successfully compiled and verified. 
This is part of a larger system for crowd safety and health monitoring.

Future Scope
- Integration with sensors (pulse oximeter, GPS)
- Cloud-based monitoring
- ML-based anomaly detection
